# Лабораторна робота №2

## Тема
Операції з примітивними типами даних. Операції з масивами.

## Завдання
Знайти найбільший елемент масиву табличних значень функції cos(x),
x - згенероване псевдовипадкове число з рівномірним розподілом у діапазоні від [0;1)
